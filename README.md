# About
“MyNotes” is an android application to write a note. Users who have an android phone  can write notes through this application.

# Features
- Write Notes
- Search Specific Notes
- Update the Note
- Delete
- Add Weblink
- Add image

# Tools used
- Room Database
- Kotlin
- XML
- Android Studio IDE
- Android version 4.0 or later
